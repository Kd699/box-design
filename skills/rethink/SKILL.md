name: rethink
description: Pause before implementing to list simpler alternatives and catch over-engineering
---

Activate when I press: /rethink

Before implementing anything, pause and ask:

1. **List 3 simpler alternatives** to the current approach (fewer lines, less setup, less complexity). For each, give a one-line tradeoff.

2. **Ask the user**: "Are we overthinking this? Which approach do you want, or should I proceed with the original plan?"

3. **Wait for user confirmation** before any implementation.

Goal: Catch over-engineering early. Prefer native APIs, fewer files, less config.
